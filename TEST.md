## 🐛 已知问题

1. **本地服务器依赖**：由于浏览器安全限制，无法直接打开本地 HTML 文件（file:// 协议），必须使用 HTTP 服务器。
   - **解决方案**：已启动 Python HTTP 服务器（端口 8000）

2. **marked.js 版本**：CDN 链接可能因版本变化而失效。
   - **解决方案**：检查浏览器控制台，如有错误，更新 CDN 链接

3. **highlight.js 主题**：当前使用 `atom-one-dark` 主题，可能需要调整。
   - **解决方案**：在 `article.html` 中更改 CSS 链接

## 🚁 部署到生产环境

如果要部署到生产环境（如 GitHub Pages、Netlify、Vercel），需要注意：

1. **生成 `articles.json`**：确保所有文章元数据都在 `articles.json` 中
2. **优化图片**：压缩图片，使用 WebP 格式
3. **CDN 优化**：将 CSS/JS 库改为生产环境 CDN 链接
4. **SEO 优化**：添加 meta 标签、Open Graph 标签
5. **分析工具**：添加 Google Analytics 或类似工具
