# AI 应用与工具新品：omlx、SuperSplat、easy-vibe、OpenClaw 等

## 🍎 omlx：Apple Silicon 本地 LLM 推理新选择

开发者 jundot 发布 omlx——专为 Apple Silicon 优化的高性能 LLM 推理服务器。核心特性：

- **连续批处理**（高效并发请求）
- **SSD 分级缓存**（热 KV 缓存驻留 RAM + 冷缓存落盘）
- **macOS 菜单栏管理界面**
- 提供 OpenAI 兼容和 Anthropic 兼容两种 API 端点

[查看原文](https://github.com/jundot/omlx)

## 🎨 PlayCanvas SuperSplat：浏览器端 3D 高斯泼溅编辑器

PlayCanvas 开源 SuperSplat Editor——基于 Web 技术的免费 3D Gaussian Splatting 编辑器，支持检查、编辑、优化和发布 3DGS 模型。引擎无关设计使其可适配各类渲染引擎，已支撑"扫描场景→生成碰撞体→制作 FPS 游戏"的完整工作流演示。

[查看原文](https://github.com/playcanvas/supersplat)

---

## 📚 Datawhale easy-vibe：面向 Vibe Coding 时代的编程入门教程

Datawhale 联合清华深圳研究生院发布 easy-vibe 开源课程，专为 AI 时代的编程初学者设计。核心理念：**"不教语法和算法，教如何用自然语言描述需求然后用 AI 工具变成产品"**。包含贪吃蛇等 AI 原生小游戏实战项目。

[查看原文](https://github.com/datawhalechina/easy-vibe)

---

## 🕵️ CloakBrowser：源码级指纹修补的反检测浏览器

CloakHQ 开源 CloakBrowser——通过 49 项 C++ 源码级补丁直接修改 Chromium 二进制指纹，30/30 项行业标准机器人检测全部通过。作为 Playwright 的 Drop-in 替代方案，适用于需要绕过反爬虫检测的自动化测试和数据采集场景。

[查看原文](https://github.com/CloakHQ/CloakBrowser)

## 📰 Digg 转型 AI 新闻聚合器 Beta 上线

历经多次易主和沉寂后，Digg 以全新面貌回归——AI 驱动的垂直领域新闻聚合器。核心逻辑：追踪特定领域中（如 AI）最具影响力的声音和权威来源，用 AI 筛选推荐真正值得关注的"高信噪比"内容。

[查看原文](https://www.fastcompany.com/91540767/digg-is-back-again-this-time-as-an-ai-news-aggregator)

## 🤗 Hugging Face x AWS 基础模型构建模块发布

## 🔄 Quick Share 将兼容苹果 AirDrop 协议

Google 宣布 Quick Share 将在今年内兼容 Apple AirDrop 协议，首批支持三星/Oppo/OnePlus/Vivo/Xiaomi/Honor 设备。过渡期可通过 Cloud QR Bridge 实现 iOS-Android 文件分享，后续还将支持密码/照片/消息/App/通讯录/eSIM 配置等完整迁移方案。
